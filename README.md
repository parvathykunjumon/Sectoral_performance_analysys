📊 Sectoral Analysis of Economic Data
📌 Overview
This project focuses on analyzing sector-wise economic data to identify growth trends, industry contributions, and key patterns. The analysis is performed using Python with data cleaning, exploratory data analysis (EDA), and interactive visualizations.
🎯 Objectives
Analyze economic growth across industries over time
Identify top contributing sectors
Study relationships between economic indicators
Detect trends, patterns, and anomalies
📂 Dataset
The dataset contains:
Industry and Sub-industry details
Economic indicators (GVA, GFCF)
Current Price and Constant Price values
Year and Quarter information
⚙️ Data Preprocessing
Handled missing values
Removed duplicates
Corrected data types
Created derived columns (Growth Rate, Year)
Performed data validation and transformation
📊 Exploratory Data Analysis (EDA)
Univariate, bivariate, and multivariate analysis
GroupBy operations and pivot tables
Correlation analysis and statistical summaries
📈 Visualizations
Created using Matplotlib, Seaborn, and Plotly:
Line charts (economic growth trends)
Bar charts (sector contribution)
Histograms (distribution analysis)
Box plots (growth variability)
Scatter plots (price relationships)
Heatmaps (correlation & trends)
Interactive charts (Plotly)
🔍 Key Insights
Identified dominant industries contributing to economic growth
Observed variability in growth rates across sectors
Found strong correlation between current and constant prices
Detected fluctuations and anomalies in certain industries
🛠️ Tools & Technologies
Python (Pandas, NumPy)
Matplotlib, Seaborn, Plotly
Jupyter Notebook
